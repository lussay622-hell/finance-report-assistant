# 财报解读助手（李宁 vs 安踏）

一个参考 `readbiz.app` 交互方式的静态前端原型，支持查看 2024-2026 年李宁与安踏的财务对比与自动解读。

## 快速使用

1. 打开项目目录：`finance-report-assistant`
2. 直接双击 `index.html`，或使用本地静态服务打开
3. 在首页输入代码（如 `2331,2020`）并点击“查看报告”

## 功能点

- 首页支持代码/公司名别名识别：`2331 / 李宁 / li-ning`、`2020 / 安踏 / anta`
- 支持快捷选择公司、请求新增 ticker（前端模拟）
- 对比页支持切换公司与年份（2024/2025/2026）
- 三大报表维度对比：
  - 利润表（收入、增速、毛利率、经营利润率）
  - 资产负债表（资产负债率、库存周转天数、净利润）
  - 现金流（经营现金流、净利现金比、现金转化率）
- 自动生成对比结论

## 目录结构

- `index.html`：首页
- `report.html`：财报对比页
- `styles.css`：全局样式
- `data.js`：示例数据（李宁、安踏，2024-2026）
- `app.js`：首页交互
- `report.js`：对比页渲染与结论逻辑
